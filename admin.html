How the data system works

Announcements published through this admin panel are saved to this browser's localStorage — they're visible on this computer immediately, but won't appear on other devices or after clearing browser data.

To make an announcement permanent (visible to all visitors), copy the exported JS below and paste it into the announcements-data.js file, then re-deploy the site. In the future this can be connected to a live database (Supabase, Firebase) to skip this step.

Click "Generate Export" to see the data file content.
